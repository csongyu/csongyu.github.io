如果 vdb不存在，则给出提示信息 this disk is not exist

使用魔法变量

最常用的有四个：

hostvars

包含受管主机的变量，可以用于获取另一台受管主机的变量的值。

group_names

列出当前受管主机所属的所有组。

groups

列出清单中的所有组和主机。

inventory_hostname

包含清单中配置的当前受管主机的主机名称。

ansible localhost -m debug -a 'var=hostvars["localhost"]'
